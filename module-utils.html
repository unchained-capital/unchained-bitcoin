(static) validateHex(inputString) → {string}

Validate whether the given string is hex.

  • Valid hex consists of an even number of characters 'a-f,A-F, or0-9`. This is case-insensitive.

  • The presence of the common prefix 0x will make the input be considered invalid (because of thex`).
